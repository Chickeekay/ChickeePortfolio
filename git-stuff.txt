Create a Repository on GitHub
Sign into GitHub, and we'll create a new repository by clicking the + icon near the top right. Select New Repository, and name it Blocbox.

Within Terminal, we'll type:

$ git init
$ git add .
$ git status
$ git commit -m "First commit"
$ git remote add origin https://github.com/<username>/Blocbox.git
$ git push -u origin master

push to repo 

$ git status
$ git add .
$ git commit -m "Design tools completed"
$ git push origin master
